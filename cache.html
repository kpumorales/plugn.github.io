Google Chrome:
Ctrl+Shift+Del → Очистить кэш → Очистить историю
Ctrl+Shift+Del (Mac: ⌘-Shift-Backspace) → Clear browsing data
Internet Explorer:
Ctrl+Shift+Del → Временные файлы Интернета → Удалить
Ctrl+Shift+Del → Temporary Internet files → Delete
Mozilla Firefox:
Ctrl+Shift+Del → Кэш → Очистить сейчас
Ctrl+Shift+Del → Cache → Clear now
Safari:
Ctrl+Alt+E → Очистить
Ctrl+Alt+E → Empty cache (Mac: ⌘ → Privacy → Remove All Website Data)
Opera:
Ctrl+F12 → Расширенные → История → Дисковый кэш → Очистить
Ctrl+F12 → Advanced → History→ Disk cache → Empty Now
